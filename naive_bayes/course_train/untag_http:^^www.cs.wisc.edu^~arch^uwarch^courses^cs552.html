CS552 Page CS/ECE 552 - Introduction to Computer Architecture 3 credits. The design of computer systems and components. Processor design, instruction set design, and addressing; control structures and microprogramming; memory management, caches, and memory hierarchies; interrupts and I/O structures. Prerequisites: ECE/CS 352 and CS/ECE 354 ; co-req: CS 367. Semesterly course information: No info as of Fall 1994.
